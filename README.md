About Opauth
------------
Opauth is a multi-provider authentication framework for PHP.

How to run
----------
1. `git submodule init`

2. `git submodule update`

3. `phpdoc run -d opauth/lib/Opauth/ -t ./ --title Opauth --sourcecode`
